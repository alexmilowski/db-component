The DocBook Web Component

Introduction

This site is a demonstration of the DocBook Web Component via the github project db-component.

Demonstrations

The Web component provides basic facilities such as a table of contents, navigation aids, cross references, and figures. It current functions correctly in Chrome. While platform.js is included to provide support in other browsers, it seems to have problems in various browsers.
